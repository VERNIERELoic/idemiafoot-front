# 🚀 Run

1. ### Clone repository

```bash
git clone https://github.com/VERNIERELoic/idemiafoot-front.git
```

2. ### Install dependencies

```bash
cd idemiafoot-front
npm install
```
3. ### Run

```bash
ng serve --configuration development # or production
```

# 🛠️ Build

```bash
ng serve --configuration production
```
